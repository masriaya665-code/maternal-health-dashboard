# Maternal Health Analytics Dashboard

An interactive Streamlit dashboard exploring global and clinical maternal health data, built for MSBA382 - Healthcare Analytics at AUB's Olayan School of Business.

**Live app:** https://maternal-health-dashboard-pmu2wupa6nfyvcrmtthxtn.streamlit.app/

## What it does

Combines WHO maternal mortality data (185+ countries, 1985-2023) with a real patient clinical dataset to give a six-page view of the problem - global trends, a geographic map, regional and income group breakdowns, a patient explorer, and a machine learning risk predictor.

## Data sources

- WHO MNCAH Data Portal - Maternal Mortality Ratio
- Kaggle / UCI ML Repository - Maternal Health Risk Dataset

## Built with

Python, Streamlit, Plotly, Pandas, Scikit-learn
